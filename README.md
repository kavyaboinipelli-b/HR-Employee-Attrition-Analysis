# HR Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition using the IBM HR Analytics Employee Attrition dataset containing 1,470 employee records and 35 features.

## Objectives
- Identify departments with highest employee attrition
- Analyze attrition across job roles
- Compare salary levels of employees who left versus stayed
- Create visualizations to support HR decision-making

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- SQL (MySQL)
- Jupyter Notebook
- GitHub

## Dataset
IBM HR Analytics Employee Attrition Dataset

## SQL Analysis
### Department-wise Attrition
Research & Development recorded the highest attrition count, followed by Sales.

### Job Role Attrition
Laboratory Technicians and Sales Executives showed the highest attrition levels.

### Salary Analysis
Employees who left the company had a lower average monthly income compared to employees who stayed.

## Python Visualizations
- Employee Attrition Count
- Attrition by Department
- Attrition by Job Role
- Average Salary by Attrition Status
